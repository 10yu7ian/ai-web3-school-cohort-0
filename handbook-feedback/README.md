# Handbook 反馈

此目录用于提交对 [AI × Web3 School Handbook](https://aiweb3.school/zh/handbook/) 的反馈、建议与勘误。

## 反馈流程

1. **发现问题** — 在阅读 Handbook 时标记疑问、错误或改进建议
2. **记录反馈** — 复制下方模板，创建新文件 `feedback-XXX.md`
3. **提交勘误** — 通过 GitHub Issues 或 PR 提交到本仓库
4. **跟进处理** — 标注反馈状态（待处理 / 已采纳 / 已回复）

## 反馈分类

| 类型 | 说明 | 标签 |
|------|------|------|
| 纠错 | 文档错误、链接失效、代码 Bug | `errata` |
| 建议 | 内容改进、结构优化、新增主题 | `suggestion` |
| 疑问 | 概念不理解、需要进一步解释 | `question` |
| 翻译 | 翻译不准确、术语不规范 | `translation` |

## 反馈模板

复制 [`feedback-template.md`](feedback-template.md) 为 `YYYY-MM-DD-简短标题.md`。

**每条 feedback 尽量包含：**

1. Handbook 页面完整链接  
2. 问题描述（卡在哪、为何困惑）  
3. 建议改法（可选）  
4. 来源（自学 / WCB / 讨论）

公开 repo 可被索引与拉取分析，便于 Handbook 共建回流。
